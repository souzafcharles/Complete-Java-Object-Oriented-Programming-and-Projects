# Example Problem 02
**Instructions**:
Write a Java program to read an integer N and the data (name and price) of N Product. Store the N products in an array. Then, display the average price of the products."

To solve this problem, you should create a CLASS according to the project alongside:

| **Product**                                             |
|---------------------------------------------------------|
| - Name: `String`                                        |
| - Price: `double`                                       | 
| =========================================               |
| + `averagePriceOfProducts(products: Product[]): double` |

***

### Example Inputs and Outputs

| **Input** | **Output**             |
|-----------|------------------------|
| 3         | AVERAGE PRICE = 700.00 |
| TV        |                        |
| 900.00    |                        |
| Fryer     |                        |                                                                                                              
| 400.00    |                        |
| Stove     |                        |
| 800.00    |                        |
