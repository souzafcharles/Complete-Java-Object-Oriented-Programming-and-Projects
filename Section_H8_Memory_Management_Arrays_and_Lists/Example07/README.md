# Exercise 14
**Instructions**:

Write a program to read an integer N and an N x N matrix of integers. Then, display the main diagonal and the count of negative values in the matrix.

### Example Inputs and Outputs

| **Input** | **Output**            |
|-----------|-----------------------|
| 3         | `Main diagonal`:      |
| 5 -3 10   | 5 8 -4                |
| 15 8 2    | `Negative numbers`: 2 |
| 7 9 -4    |                       |
