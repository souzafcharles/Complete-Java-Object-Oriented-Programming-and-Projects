# Example Problem 01
**Instructions**:
Write a program to read an integer N and the height of N people. Store the N heights in an array. Then, display the average height of these people.

### Example Inputs and Outputs

| **Input** | **Output**            |
|-----------|-----------------------|
| 3         | AVERAGE HEIGHT = 1.69 |
| 1.72      |                       |
| 1.56      |                       |
| 1.80      |                       |                                                                                                              |
